# MStore Server v0.1.0

Release date: 2026-03

## Changes

- Initial release
- S3 HTTP endpoint (port 9010)
- gRPC endpoint (port 9011)
- Erasure coding with Reed-Solomon
- SSE-S3, SSE-KMS, SSE-C encryption
- Object versioning and delete markers
- WORM object lock (SEC 17a-4)
- Multipart uploads
- Web console
- Full-text search
- Background healing
- Site replication (async/sync)
