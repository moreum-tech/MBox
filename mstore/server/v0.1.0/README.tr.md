# MStore Sunucu v0.1.0

Yayın tarihi: 2026-03

## Değişiklikler

- İlk sürüm
- S3 HTTP uç noktası (port 9010)
- gRPC uç noktası (port 9011)
- Reed-Solomon ile erasure coding
- SSE-S3, SSE-KMS, SSE-C şifreleme
- Nesne sürümleme ve silme işaretçileri
- WORM nesne kilidi (SEC 17a-4)
- Çoklu parça yükleme (multipart)
- Web konsolu
- Tam metin arama
- Arka plan onarımı
- Site replikasyonu (asenkron/senkron)
